# Simple JS Snake Game

A minimalist browser-based Snake game created with JavaScript and HTML5 Canvas.

## How to Play
1. **Start:** Open the HTML file in any modern web browser.
2. **Move:** Use the **Arrow Keys** (Up, Down, Left, Right) to control the snake.
3. **Objective:** Eat the **yellow food** to grow your snake.
4. **Avoid:** Do not hit the walls or your own tail.
5. **Reset:** Click the "Reset" button to start a new round after a Game Over.

## Built With
* **HTML5 Canvas:** For rendering the game graphics.
* **JavaScript (ES6):** For game logic, movement, and collision physics.
* **CSS:** For basic layout and canvas styling.
